# Sarah Liaw — Personal Website

My personal academic website, built with **Jekyll** and hosted on **GitHub Pages** at [sarahliaw.github.io](https://sarahliaw.github.io).

## Structure

- `_layouts/default.html` — Main page layout (hero, news, publications, projects)
- `_layouts/post.html` — Detail page for individual publications
- `_posts/` — Research publications (each gets its own detail page)
- `images/` — Profile photo and paper thumbnails
- `style.scss` — All styling (dark theme, Inter font, responsive)
- `_config.yml` — Jekyll configuration

### New News Item
Edit `_layouts/default.html` and add a `<li class="news-item">` entry to the news section.

### New Project
Edit `_layouts/default.html` and add a `<a class="project-card">` entry to the projects section.
